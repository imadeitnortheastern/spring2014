What's a Port?


Imagine a computer's IP address is its phone number and the port is like an extension. Just like an office can have one phone number and hundreds of extensions to reach individual employees, we are using iMadeIT's IP as the base phone number, and giving you each an extension. This way, whenever you want to show off what you're working on, you can just point your friends to imadeit.nu:your port number

How Do I Access My Port?


Mac/Linux
You can start by firing up a terminal shell by searching your applications for 'terminal'. In the window that opens simply type

ssh your_user_name@imadeit.nu

When it asks you for a password, simply type your password for this site. Any further authorization questions you can answer simply by typing 'yes' and pressing enter.

That's it! You're in! You can get a Flask app up and running by creating a Flask file like we'll describe in class, but the last line should look like:

app.run(host='0.0.0.0', port=your_port_number_here)


Windows
If you have Windows, you can will need to install PuTTy or some other SSH client on your computer.

In the address field you should type your_user_name@imadeit.nu and leave the port set to 22 (this is the port on computers that listen for SSH connections) and click connect.

That's it! You're in! You can get a Flask app up and running by creating a Flask file like we'll describe in class, but the last line should look like:

app.run(host='0.0.0.0', port=your_port_number_here)
