Getting Your Own App Up On iMadeIT.nu


The first step is SSHing into the iMadeIT server. There are instructions over at The Port Authority explaining how to SSH depending on your OS, but the idea is the same regardless; you are remotely logging into a computer that you will load your application onto. The steps for getting your wep app up are almost perfectly analogous to getting an app up and running on your own computer. The only difference is on the last line of your flask app, where you had the line:

app.run()

This line only runs the app locally on your own computer, and if you leave it like that, it will only run it locally on the server. If you want to get make it visible to the outside world, you need to set the host argument to be globally visible, which is accomplished by setting host='0.0.0.0'. The only step left is to make sure that the port on the server that your are listening on isn't being used by someone else. If you haven't selected a port yet, go ahead and visit The Port Authority and selecting one. If you have, you just need to set the port argument equal to your port (port=your_port).

So for example, with my port number being 6478, my last line is:

app.run(host='0.0.0.0', port=6478)

And with my app being called hello.py, I can get my app up and running by typing:

python hello.py &

To visit your page, all you need to do is visit iMadeIT.nu/your_user_name/your_app_route So, with my username being jamesmc, and my flask app having instructions for @app.route('/intro') I would visit:

iMadeIT.nu/jamesmc/intro

Installing Flask and Python


Hi future programmers! If you plan on coming to our first meeting (102 WVG 7-9 PM on Thu, Jan 23), PLEASE try to have Python and Flask installed on your computer before you come. If you have ANY problems at all, don't hesitate to reach out to us, or coming to the first meeting a bit early.


MAC/LINUX

You most likely already have Python 3.3 on your computer. To check, open the application browser, type in "terminal" and run the application displayed. In the application, type the word "python" (no quotes) and hit enter. If you see some text describing a python version, and the blinking cursor is in front of three "greater-thans" (>>>) than you can skip to installing Flask.

Installing Python
Python is the programming language we will be using through out the course.

First download Python 3.3
Once the file has been downloaded, open the file and run the installer, following the on-screen instructions to complete installation.

Installing Flask
Flask is the web-framework we will be using; it is a python program that has lots of tools that make it supa easy to build cool websites.

Follow theseinstructions to install Flask and Virtualenv. When it asks you to run commands, you can do this by opening the application browser, typing in "terminal" and running the application displayed, as above. Any command can simply be typed into the application that opens.

That's it, you're done! See you at iMadeIT!
