Installing Flask and Python


Hi future programmers! If you plan on coming to our first meeting (102 WVG 7-9 PM on Thu, Jan 23), PLEASE try to have Python and Flask installed on your computer before you come. If you have ANY problems at all, don't hesitate to reach out to us, or coming to the first meeting a bit early.


MAC/LINUX

You most likely already have Python 3.3 on your computer. To check, open the application browser, type in "terminal" and run the application displayed. In the application, type the word "python" (no quotes) and hit enter. If you see some text describing a python version, and the blinking cursor is in front of three "greater-thans" (>>>) than you can skip to installing Flask.

Installing Python
Python is the programming language we will be using through out the course.

First downloadPython 3.3
Once the file has been downloaded, open the file and run the installer, following the on-screen instructions to complete installation.

Installing Flask
Flask is the web-framework we will be using; it is a python program that has lots of tools that make it supa easy to build cool websites.

Follow theseinstructions to install Flask and Virtualenv. When it asks you to run commands, you can do this by opening the application browser, typing in "terminal" and running the application displayed, as above. Any command can simply be typed into the application that opens.

That's it, you're done! See you at iMadeIT!
